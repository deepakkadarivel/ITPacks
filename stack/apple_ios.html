Apple iOS

iOS is a mobile operating system created and developed by Apple Inc. exclusively for its hardware. It is a proprietary mobile operating system that runs on the iPhone, iPad and iPod Touch.

    iOS certified expert must be able to use iOS and perform the following activities such as:
  • UITableView Touch Up
  • Layout with UICollectionView
  • Creating a contact detail page
  • Creating animation
  • Avoiding complex inheritance with protocols
  • Adding core data to the app
  • Storing and querying data in core data
  • Fetching and displaying data from the network
  • Enriching apps with the camera, motion and location
  • Displaying contents of the app in spotlight
  • Making the web and the app meet through universal links
  • Instant information with a notification center widget
  • Implementing rich notifications
  • Extending iMessage
  • Ensuring app quality with tests

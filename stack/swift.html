SWIFT

Swift is a powerful and intuitive programming language for macOS, iOS, watchOS and tvOS. It is a general-purpose programming language built using a modern approach to safety, performance, and software design patterns.

    SWIFT certified expert must be able to use SWIFT language and perform the following activities such as:
  • Using Collections and Cocoa Data Types - Arrays; Dictionaries
  • Control Flow and Functions - The for loops
  • Creating a class or structure - Initializer; Protocols; Memory management
  • Working with XML and JSON Data
  • Custom Subscripting
  • Using Optional Type and Optional Chaining
  • Working with Generics
  • Working with Closures
  • Concurrency and parallelism
  • Creating and managing dispatch queues
  • Using NSOperation and NSOperationQueue types
  • Swift Formatting and Style Guide
  • Network Development with Swift

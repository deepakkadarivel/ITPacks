CERTIFIED ASSOCIATE IN WEB DEVELOPMENT
(CAWD)

  • HOME /
  • CERTIFICATES /
  • CERTIFIED ASSOCIATE IN WEB DEVELOPMENT (CAWD)

The CAWD certification is primarily for new web developers. Creating web applications is a complicated task involving lots of moving parts and interacting components. In order to become a developer of web applications, you will have to be well versed in at least four languages: HTML, CSS, JavaScript, and a server-side language.

PRE-REQUISITES/ELIGIBILITY:

Minimum 80 hours of education in Web development.
Education could be through learning institutes, boot camps, colleges, e-learning. Proof of learning is required to be submitted as part of the application.

DOMAINS (SYLLABUS):

  • Design web applications in accordance to requirements
  • Build web applications with interactive page elements
  • Fetch, update and store data from the web, cloud or database
  • Perform tests

Context:

Test takers can choose the concepts and technologies on which to be tested. The selection has to comprise of what is needed for bare minimum full stack development including front-end and back end tools. The exam will be customized based on the selection. Various front-end and back-end concepts and languages available for testing as part of the CAWD exam include but not limited to:

  • JavaScript
  • HTML
  • CSS
  • jQuery
  • AJAX
  • Command Line
  • Node.js
  • Java
  • Python
  • SQL
  • Angular.js
  • D3.js
  • Express.js
  • MongoDB
  • APIâ€™s
  • Any other frameworks, languages, tools

EXAM DETAILS:

  • Duration: 120 minutes
  • Format: Online Supervised/Recorded
  • Number of Questions: 30. (MCQs, Written questions, Exercises)
  • Result: Within 2 weeks from exam
  • Passing Score: 65% (Questions will carry different weightage)

TARGET AUDIENCE:

  • Entry level web developers
  • Programmers interested in Full Stack web development
  • Software engineers

Proof of Concept:

Non-required

MAINTAINING PDUS:

To maintain your credential, you will need to accumulate 30 Professional Development Units (1 PDU = 1 hr of training) over 2 years.

APPLICATION STEPS:

  1. Choose your set of full stack technologies
  2. Complete exam application
  3. Make exam fee payment
  4. Complete audit process (if selected for audit)
  5. Receive eligibility notification
  6. Choose date for exam
  7. Take exam

Test Re-Take:

In case you fail the exam, you will be provided with a report highlighting areas for improvement to help you prepare better for the next attempt. You can take the CAWD exam a maximum of 3 times in a year. With each attempt you will need to pay the full amount of the first attempt exam fee. Re-take fee is waived off if you have obtained your education hours through one of our Registered Training Providers (RTPs).

SAMPLE INDUSTRY ROLES SUITED FOR A CAWD :

  • Associate Web Developer
  • Junior Web Designer

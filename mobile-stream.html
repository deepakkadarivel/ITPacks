Mobile Development

Mobile application development is the set of processes and procedures involved in writing software for small, wireless computing devices such as smartphones or tablets.

These applications may be pre-installed on phones during manufacturing platforms, or delivered as web applications using server-side or client-side processing to provide an ‘application-like’ experience within a web browser. Mobile application software developers must consider a long array of screen sizes, hardware specifications, and configurations. Mobile user interfaces rely on mobile back-ends to support access to enterprise systems. The mobile back-end facilitates data routing, security, authentication, authorization, working off-line, and service orchestration.
